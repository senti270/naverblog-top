개발가이드

프로젝트 개요

지점별 키워드 리스트를 저장하고, 각 키워드에 대한 네이버 블로그 상위 3개 URL을 조회하는 웹 애플리케이션입니다.

주요 기능

  • 지점별 키워드 관리 (추가/삭제)
  • 네이버 블로그 API를 통한 상위 3개 URL 조회
  • 유사도(sim) 정렬 고정
  • 새 창으로 URL 열기

기술 스택

  • Backend: FastAPI + Python 3.11+
  • Database: SQLite
  • Frontend: Vanilla JavaScript + HTML/CSS
  • External API: 네이버 블로그 검색 API

API 엔드포인트

지점 관리

GET /api/branches

모든 지점 목록을 조회합니다.

키워드 관리

GET /api/keywords?branch_id={id}

특정 지점의 키워드 목록을 조회합니다.

POST /api/keywords/add

새 키워드를 추가합니다.

{ "branch_id": 1, "keyword": "새 키워드" }
DELETE /api/keywords/delete

키워드를 삭제합니다.

{ "branch_id": 1, "keyword": "삭제할 키워드" }

블로그 조회

POST /api/run

저장된 키워드로 네이버 블로그 상위 3개 URL을 조회합니다.

{ "branch_id": 1 }

환경 설정

필요한 환경변수

NAVER_ID=your_client_id_here NAVER_SECRET=your_client_secret_here

네이버 API 설정

  1. 네이버 개발자 센터에서 애플리케이션 등록
  2. 블로그 검색 API 사용 설정
  3. Client ID와 Client Secret 발급
  4. .env 파일에 환경변수 설정

배포 방법

Railway 배포 (추천)

  1. GitHub에 코드 푸시
  2. Railway.app에서 계정 생성
  3. "New Project" → "Deploy from GitHub repo" 선택
  4. 저장소 선택 후 자동 배포
  5. 환경변수 설정 (NAVER_ID, NAVER_SECRET)

Docker를 사용한 배포

# 이미지 빌드 docker-compose build # 서비스 시작 docker-compose up -d

수동 배포

# 의존성 설치 pip install -r requirements.txt # 서버 실행 uvicorn app:app --host 0.0.0.0 --port 8000
